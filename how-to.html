アプリの開き方

次の URL から Web アプリを開きます（スマートフォン・PC どちらでも利用できます）。

https://app.ai-trading-signal.com/

基本的な操作

  1. 左サイドバーで商品種別・セクターなどを選び、「絞り込み検索」を押す
  2. 銘柄一覧から銘柄を選ぶ、またはチェックを入れて「選択した銘柄を表示」
  3. 日足チャート上の AI 売買目安ラインと達成状況を確認する

ご注意

  • 表示内容は参考情報であり、投資助言ではありません
  • チャートデータは外部 API 等から取得しており、遅延・欠損がある場合があります
  • 管理者向け機能（URL に ?admin=1 等）は一般公開 URL では利用しないでください
